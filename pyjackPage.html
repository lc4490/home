PyJack

This was my first personal Python project, and runs a blackjack game in the terminal. The program allows the user to login with a pre-existing username and password, or create a new account. Each account has a card, and the user is prompted to deposit value into the card (functions like a casino). The program has two difficulties, one AI dealer that will always hit before 17 and never hit after, and another dealer that only hits if their card total is less than the player’s. The blackjack program is fully equipped with hit, stand, and split functions. After playing, the program stores the user’s information along with their current card total.

This was my very first personal project, and I was inspired by twitch streamer Ludwig’s working blackjack casino in Minecraft. We had just learned about crafting a deck of card-like data structure in class, so I figured I could create a blackjack program of my own. It initially started off with just printing the card information like [5, Diamond], [6, Clubs], but I created a function to print out some ASCII art to represent the cards. I also eventually added the user interface as well as multiple difficulties.

To further improve this project, I would try and handle more exceptions, as the program is at the mercy of the user, and will not accept inputs other than the ones provided. I have also considered turning this project into an app or web game.
