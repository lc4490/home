Macro Tracker

This Python GUI was written for my sister, who wanted an app to keep track of her meals during her diet. The program allows the user to add recipes, add ingredients to each recipe, and add macronutrient ratios to each ingredient. The program will store saved recipes in txt files on the computer, and will recall recipe information when the program is re-opened.

This project made me familiar with the tkinter class, as I learned how to use labels, buttons, input boxes, etc. It taught me how to manipulate different frames to create different pages in the GUI. I also learned how to detect whether a computer is light/dark mode, so I could make my program light/dark mode compatible.

To further improve this project, I would try to add scrollbars so that the user does not have to expand the window to access more recipes. I also have the idea of adding a calendar to the application, in order for the user to track their meals throughout the week.
