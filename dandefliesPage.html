Dandeflies

This P5 project was written for my creative coding class. It uses a for loop to create a circle of butterfly objects and dandelion strands. When the user clicks on the screen, the butterflies will fly in the away from the position the mouse clicked, and will continue to do so until they all fly off screen. The program will restart when the user presses the space button.

This project taught me how to use the P5 library in Javascript. I learned how to use mouse functions as well as keyboard command functions. I also learned how to use translate, rotate, scale, etc. functions in order to manipulate the image. I was already familiar with object oriented programming, but this project help me apply it and understand it from a new angle.

To further improve this project, I would perhaps add color to the butterflies or add some type of random pattern to make it look more interesting.
